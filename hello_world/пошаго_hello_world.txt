************************************************** ЗАДАЧА 1 ****************************************

1. создаём проект (папка например app_project)

2. запускаем терминал и подключаем виртуальное окружение
python -m venv venv

3. активируем виртуальное окружение
venv\Scripts\activate

4. устанавливаем джанго
pip install django

5. обновляем pip 
python -m pip install --upgrade pip

6. проверяем наличие пакетов
pip list

7. Создаем проект джанго в конце через пробел ставим точку
django-admin startproject main .

8. запускаем джанго 
python manage.py runserver

9. тушим сервер
ctr + c

8. проверяем наличие команд manage.py
python manage.py

9. применяем миграцию моделей уже имеющихся по умолчанию в джанго
python manage.py migrate

10. создаём приложение
python manage.py startapp hello_world

11. создаём в hello_world файл urls.py

12. в файл app/settings.py вносим изменения
*********************
INSTALLED_APPS = [
    ... ,
    ... ,

    'hello_world',    
]

LANGUAGE_CODE = 'en-us'
на
LANGUAGE_CODE = 'ru-ru'

TIME_ZONE = 'UTC'
на
TIME_ZONE = 'Europe/Moscow'
******************

13. открываем main_app/views.py делаем представление модели
***********
from django.shortcuts import render
from django.http import HttpResponse


def homepageview(request):
    return HttpResponse('Hello, World!')
***********

14. открываем main/urls.py вносим изменение в URL не забываем импортировать include
***************
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('hello_world.urls')),
]
**************

15. открываем main_app/urls.py вносим изменение в URL не забываем импортировать views из main_app
***************
from django.urls import path
from hello_world import views

urlpatterns = [
    path('', views.homepageview, name='homepageview'),
]
***************

16. запускаем сервер проверяем
python manage.py runserver
